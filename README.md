# fluent-top

fluent-top is a top-like command line utility to display Fluentd status.

fluent-top is based on [resque-top](https://github.com/miyagawa/resque-top)

## Installation

### gem

```
gem install fluent-top
```

## Usage

```
fluent-top
```
